# Tailwind portfolio

Simple personal portfolio build with [Tailwind CSS](https://tailwindcss.com/).

## Run project

- Clone the project
```bash
  git clone https://github.com/Josephglz/PortfolioUpdated.git

  cd JosephGlz
```

- Install dependencies & start the server
```bash
  npm install

  npm run dev
```
